Welcome

openEduConnect is a place to build and contribute to open source tools for education data. Futhermore, it is also a knowledge sharing network for those involved in the field.

Projects

Our first project is Eduextractor, a Python package for extracting and transforming data from locked down educational data platforms.

Futhermore, contributors work on projects such as Code for America's Somerville Teacher Tool and MapVizeR, an R package for visualizing NWEA MAP exam data.

History

A few friends who happen to work in open data & technology discover they are both working on projects with school systems.

The school systems both want to export data from their Student Information Systems. They want to use that data to build tools that let them find new insights and take actions that support students.

The friends started to wonder:

"If this is such a common use case..."

And the rest, as they say, is history.
